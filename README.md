# Weather Dashboard

## Purpose
A dashboard that uses the OpenWeather API to retrieve weather data for cities that are searched for by user. 
Uses localStorage to store any cities that have been searched for prior 

## Built With
* HTML
* CSS
* JavaScript
 
  Libraries:
  * jQuery
  * Moment.js
  * Bootstrap
